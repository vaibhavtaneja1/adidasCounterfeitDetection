The counterfeit product detection fro Adidas brand has been built using Convolutional Neural Networks. Existing weights of VGG 16 were used and final layers were trained for our use-case.

Training Code is available in the notebook file and the model was then used in a django based web application.

The counterfeit adidas product detection application has been deployed on heroku and can be accessed at https://adidasfakelogodetector.herokuapp.com/
